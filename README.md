# QR Code Security Scanner

A Python-based QR Code Security Scanner that scans QR codes and extracts information from them.

## Features
- Scan QR codes
- Decode QR contents
- Image upload support
- Simple and user-friendly

## Technologies Used
- Python
- OpenCV
- Pyzbar

## Files
- `app.py` - Main application
- `qr_scanner.py` - QR scanning functionality
- `uploaded.png` - Sample image
